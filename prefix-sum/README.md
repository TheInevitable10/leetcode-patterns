# Prefix Sum

## 🧠 Pattern

A prefix sum stores the cumulative sum of elements from the beginning of the array up to each index.

Formula:

prefix[i] = prefix[i - 1] + nums[i]

## 📌 When to use

- Range sum queries
- Continuous subarrays
- Sum-related problems
